# Hershey Fonts

Place Hershey font files (`.jhf` format) in this directory.

## Quick Start

Copy `.jhf` files from `/Users/mgilbert/Code/hershey-fonts/hershey-fonts/` to this directory:

```bash
cp /Users/mgilbert/Code/hershey-fonts/hershey-fonts/*.jhf .
```

Or copy individual fonts:
```bash
cp /Users/mgilbert/Code/hershey-fonts/hershey-fonts/futural.jhf .
cp /Users/mgilbert/Code/hershey-fonts/hershey-fonts/scripts.jhf .
```

## Popular Hershey Fonts

- `futural.jhf` - Futura-like, clean sans-serif (great for tech)
- `scripts.jhf` - Script/cursive style (handwriting feel)
- `scriptc.jhf` - Script complex
- `cursive.jhf` - Cursive handwriting
- `timesr.jhf` - Times Roman
- `rowmans.jhf` - Roman Simplex

## Usage

1. Copy `.jhf` files to this directory
2. In WriteTyper, click "Load Font"
3. Browse to `resources/fonts/` and select a `.jhf` file
4. Start typing!

## Git Ignore

Font files (`.jhf` and `.svg`) in this directory are gitignored, so you won't accidentally commit large font files to your repo.
